# 🤖 AI Assistant Plugin System

An intelligent and modular **AI Assistant Plugin System** built in Python that supports multiple plugins like Weather, Jokes, and News.  
Designed for flexibility and scalability — new plugins can be added easily to expand the assistant’s abilities. 🌟

---

## 🧠 Features

✅ **Weather Plugin** – Fetches live weather updates for any city using real-time API.  
😂 **Joke Plugin** – Delivers a random joke to lighten your mood.  
📰 **News Plugin** – Provides the latest news headlines from trusted sources.  
⚙️ **Modular Architecture** – Each plugin works independently and can be easily replaced or extended.  
💬 **Simple Command Interface** – Just type commands like `"weather"`, `"joke"`, or `"news"` to interact with plugins.
